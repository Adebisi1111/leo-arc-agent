Concord

Autonomous USDC payroll agent — runs recurring salary payments on Arc Testnet with no manual approval per cycle.

What it does

  • Monitors payment schedules on-chain (SubscriptionVault)
  • Executes recurring USDC payroll via Circle programmable payments
  • Confirms each payment and reports it — hands-off, on schedule
  • Payroll-only: no subscriptions, no manual send button

Why it matters

  • Real Agentic-Economy use case: agents running money reliably
  • Safety-first: never pays unless it can verify a payment is due
  • Built natively on Arc + Circle, not a wrapper
  • Silent until it acts — built for autonomous operation
Proof it works: Concord executed real USDC payments on Arc testnet in July 2026 via Circle SCP. Live dashboard: adebisi1111.github.io/leo-arc-agent · Code: github.com/Adebisi1111/leo-arc-agent

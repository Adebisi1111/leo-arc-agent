OPTION 1 — USE THE DASHBOARD SEND BUTTON (needs a computer + Node)
==================================================================
This requires a laptop/desktop with Node.js (cannot run on phone).
It makes the dashboard's "Send" button actually work.

STEP A — Get the backend running on your machine
-------------------------------------------------
1. Clone the repo (or use the files already there):
   git clone https://github.com/Adebisi1111/leo-arc-agent.git
   cd leo-arc-agent

2. Install deps (one time):
   npm install

3. Create your .env file from the template:
   cp .env.example .env
   Then edit .env and fill in your real values:
     CIRCLE_API_KEY=your_real_circle_api_key
     CIRCLE_ENTITY_SECRET=your_real_entity_secret
     CIRCLE_WALLET_ID=your_agent_wallet_id_here

4. Start the server:
   node api-send.mjs
   You should see: "Concord send API on :3001"

STEP B — Expose it with a public HTTPS URL (ngrok)
--------------------------------------------------
The dashboard is on GitHub Pages (public) but the backend is on your
localhost — it needs a public URL so the browser can reach it.

5. In a SECOND terminal, run ngrok (install from https://ngrok.com):
   ngrok http 3001
   ngrok gives you a URL like: https://xxxx.ngrok-free.dev
   Copy that URL.

STEP C — Tell the dashboard that URL
-------------------------------------
6. Send me that ngrok URL. I will set docs/config.json:
     { "sendApi": "https://xxxx.ngrok-free.dev/api/send" }
   and push it. (This is the only step I do.)
